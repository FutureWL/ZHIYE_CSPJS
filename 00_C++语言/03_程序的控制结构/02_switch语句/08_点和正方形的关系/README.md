1. **问题描述**：
   - 假设有一个以原点 `(0, 0)` 为中心，边长为 `2` 的正方形。给定一个点 `(x, y)`，判断该点与正方形的关系：点在正方形内、点在正方形上（边界）或点在正方形外。我们将使用 `switch` 语句来实现这个判断。
2. **代码实现**：
```cpp
#include <iostream>

int main() {
    double x, y;
    std::cout << "请输入点的坐标 (x, y): ";
    std::cin >> x >> y;

    int position = 0;
    if (x >= -1 && x <= 1 && y >= -1 && y <= 1) {
        if (x == -1 || x == 1 || y == -1 || y == 1) {
            position = 1; // 点在正方形上
        } else {
            position = 2; // 点在正方形内
        }
    } else {
        position = 3; // 点在正方形外
    }

    switch (position) {
    case 1:
        std::cout << "点在正方形上。" << std::endl;
        break;
    case 2:
        std::cout << "点在正方形内。" << std::endl;
        break;
    case 3:
        std::cout << "点在正方形外。" << std::endl;
        break;
    }

    return 0;
}
```
3. **代码说明**：
   - **变量声明**：
     - 声明两个 `double` 类型变量 `x` 和 `y`，用于存储点的坐标。
     - 声明一个 `int` 类型变量 `position`，用于标记点与正方形的位置关系，初始化为 0。这里 `1` 表示点在正方形上，`2` 表示点在正方形内，`3` 表示点在正方形外。
   - **输入获取**：
     - 使用 `std::cout` 提示用户输入点的坐标 `(x, y)`，并通过 `std::cin` 获取用户输入的值分别存储到 `x` 和 `y` 中。
   - **确定点的位置**：
     - 首先通过 `if` 语句判断点是否在以 `(-1, -1)`，`(-1, 1)`，`(1, 1)` 和 `(1, -1)` 为顶点的正方形范围内（包括边界）。如果在范围内，再进一步判断点是否在正方形的边界上（即 `x` 等于 `-1` 或 `1`，或者 `y` 等于 `-1` 或 `1`），如果在边界上则将 `position` 设为 `1`，否则设为 `2`。如果点不在正方形范围内，则将 `position` 设为 `3`。
   - **`switch` 语句输出结果**：
     - 根据 `position` 的值进入不同的 `case` 分支。
     - `case 1` 分支对应点在正方形上的情况，输出相应信息。
     - `case 2` 分支对应点在正方形内的情况，输出相应信息。
     - `case 3` 分支对应点在正方形外的情况，输出相应信息。每个 `case` 分支后都使用 `break` 语句，防止程序继续执行下一个 `case`。

这个程序通过 `switch` 语句实现了判断点与正方形的位置关系的功能。如果需要调整正方形的位置或大小，只需修改判断条件中的边界值即可。
