# 简单算数表达式求值 
# 【1.12 编程基础之函数与过程抽象 01】

两位正整数的简单算数运算（只考虑整数运算），算术运算为：
- +，加法运算；
- -，减法运算；
- *，乘法运算；
- /，整除运算；
- %，取余运算。

算数表达式的格式为（运算符前后可能有空格）：

运算数 运算符 运算数

请输出相应的结果。

输入：

一行算数表达式。

输出：

整型算数运算的结果（结果值不一定为两位数，可能多于两位或少于两位）。

样例输入：

32+64

样例输出：

96
